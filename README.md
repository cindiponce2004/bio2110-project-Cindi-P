# bio2110-project-Cindi-P
Flu Vaccine Rate Analysis

This project analyzes flu vaccine rates among different age groups, race/ethnicities and boroughs using NYC immunization data. The goal of the project was to find  disparities in vaccination coverage among different populations. 

Research question 
How do flu vaccination rates differ by age group, race/ethnicity and borough? 

The dataset contains flu vaccination counts, age group categories, racial and ethnic classification and borough data. 

How to Run the Project 
Open the notebook in google colab.
Upload the dataset if necessary.
Run all the cells from top to bottom.

Libaries used:
numpy, matplotlib, and pandas
